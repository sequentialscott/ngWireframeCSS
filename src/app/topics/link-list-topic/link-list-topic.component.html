• You can use the class link-list on a UL element to get a single column list.
  • The CSS supports up to four columns if you add the class columns-n, where n is the number of columns.
  • You can also use the sg-link-list component, supplying it an array of objects with {{ '{' }}text: 'Link text', url: 'url text' {{ '}' }}. There is also an optional columns attribute.
