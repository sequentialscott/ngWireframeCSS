No options available for breadcrumbs widget.
  • Breadcrumbs can be structural or historical. Don't use both in the same site.
  • Structural breadcrumbs show where you are in a hierarchy.
  • Historical breadcrumbs show the path you took when navigating to the page.
  • The current page (last one in the object passed in with via the links attribute) does not have a link.
  • If an item in the linkset list doesn't have a link target, it will receive an href of "#", the current page.
