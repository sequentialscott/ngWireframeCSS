Panel Heading

The panels follow the same format as Bootstrap, in case that helps you.

You could stick a status bar down here. Or totals.
  • Use panels whenever you need a box.
  • The HTML for panels is the same as Bootstrap.
  • The color options are CSS classes based on a subset of CSS color keywords. You can use them on anything with a background color.
  • Use the class drawn on the .panel for the hand-drawn borders. You can use this class on other elements as well.
